# 写作风格指南

## 一、语言风格

### 1.1 简洁与生动的平衡

**简洁不等于简单**
- ✅ 好例子："夕阳像融化的金子，顺着屋檐缓缓流淌。"
- ❌ 差例子："太阳落山了，光线很黄很亮。"

**避免冗余表达**
- ❌ "他亲眼目睹了整个过程" → ✅ "他目睹了整个过程"
- ❌ "进行一个调查" → ✅ "调查"
- ❌ "在……的情况下" → 直接删去或改写

### 1.2 节奏把控

**句子长度变化**
```
长句：描写环境、渲染氛围、展现复杂心理
短句：制造紧张感、强调重点、加快节奏

示例：
风停了。树影凝固在地上，像一幅剪纸。
远处传来脚步声，很慢，很沉，每一步都踏在人心上。
```

**段落节奏**
- 动作场景：短段落，快速切换
- 情感场景：适中段落，留有呼吸空间
- 描写场景：长段落，沉浸体验

## 二、对话写作

### 2.1 对话三要素

1. **符合身份**：什么样的人说什么样的话
   - 学者：用词准确，可能有书卷气
   - 江湖人：干脆利落，可能带口头禅
   - 孩童：天真直接，可能有语法错误

2. **推动剧情**：每句话都要有目的
   - 传递信息
   - 展现关系
   - 制造冲突
   - 揭示性格

3. **潜台词**：话里有话
   ```
   表面："今天天气不错。"
   实际："我想和你谈谈，但不知从何说起。"
   ```

### 2.2 对话标签的使用

**少用"说"**
```
❌ "我不想去。"他说。
✅ "我不想去。"他低下头，踢着脚下的石子。
```

**动作代替标签**
```
❌ "小心！"他喊道。
✅ 他猛地扑过来，一把将她推开。
```

## 三、描写技巧

### 3.1 五感描写

不要只用视觉，调动所有感官：

```
视觉：夕阳将天空染成血红色
听觉：远处传来乌鸦的叫声，沙哑而凄凉
嗅觉：空气中弥漫着潮湿的泥土气息
触觉：冷风像刀子一样刮过脸颊
味觉：嘴里泛起铁锈般的血腥味
```

### 3.2 动静结合

**以动写静**
```
❌ 房间很安静。
✅ 墙上的挂钟滴答作响，在寂静中格外清晰。
```

**以静写动**
```
❌ 他跑得很快。
✅ 世界在他耳边凝固成一片模糊的色彩。
```

### 3.3 详略得当

**重点详写**：
- 关键转折点
- 情感高潮
- 独特设定
- 核心冲突

**次要害写**：
- 过渡场景
- 常见动作（如开门、坐下）
- 读者已知信息

## 四、视角与叙事

### 4.1 视角选择

**第一人称**：
- 优点：代入感强，心理描写自然
- 缺点：只能写"我"知道的事
- 适合：悬疑、情感、成长故事

**第三人称（限知）**：
- 优点：灵活切换，保持悬念
- 缺点：需要处理视角转换
- 适合：大多数小说

**第三人称（全知）**：
- 优点：信息量大，可评论
- 缺点：代入感弱，容易说教
- 适合：史诗、群像剧

### 4.2 展示 vs 讲述

**展示（Show）**：
```
他的手在颤抖，杯子里的水面泛起涟漪。
```

**讲述（Tell）**：
```
他很紧张。
```

**原则**：能展示就不讲述，但关键信息可以直说。

## 五、常见文体问题

### 5.1 开头通病

❌ **信息堆砌**：大量背景设定一股脑倒出
✅ **从场景切入**：让读者在故事中了解世界

❌ **慢热拖沓**：前几千字都在铺垫
✅ **快速入戏**：第一章就要有冲突或悬念

### 5.2 过渡处理

**时间跳跃**：
```
❌ 三年后……
✅ 那年的樱花又开了，只是树下的人已经不同。
```

**场景切换**：
```
❌ 与此同时，在另一个地方……
✅ 用共通的元素自然过渡（如相同的物品、相似的情绪）
```

### 5.3 结尾艺术

**章节结尾**：
- 悬念：留下未解之谜
- 情感：定格情绪高点
- 转折：突然逆转

**全书结尾**：
- 闭环：呼应开头
- 升华：主题提炼
- 留白：余韵悠长

## 六、类型小说要点

### 6.1 玄幻/仙侠
- 设定要自洽，建立清晰的力量体系
- 升级要有代价，不能过于轻松
- 世界观通过故事展现，而非说明

### 6.2 悬疑/推理
- 公平性：线索要展示给读者
- 逻辑性：推理过程严密
- 意外性：结局既在意料之外，又在情理之中

### 6.3 言情
- 感情发展有层次，不要一蹴而就
- 阻碍要真实可信
- 化学反应比外貌描写更重要

### 6.4 历史
- 大事不虚，小事不拘
- 时代感通过细节体现
- 避免现代思维强加于古人

## 七、修改清单

完成初稿后，逐条检查：

- [ ] 开头是否能吸引读者继续读下去？
- [ ] 每个场景是否推动了剧情或塑造了人物？
- [ ] 对话是否自然，符合人物身份？
- [ ] 描写是否过于冗长或过于简略？
- [ ] 是否有重复的信息或场景？
- [ ] 情感转折是否有足够的铺垫？
- [ ] 结尾是否令人满意？
- [ ] 文字是否还有精简的空间？
