## 你的任务

对原文进行润色，使其符合润色要求，同时遵循风格指南。

## 绝对规则（违反将导致失败）

1. **禁止扩写** - 只优化表达，不增加情节、描写、对话
2. **保持长度** - 润色后字数与原文相差不超过20%
3. **允许调整句长** - 原文段落数保持不变，但段内可将长句拆为短句、调整句子顺序（遵循风格指南）
4. **禁止添加** - 不添加原文没有的情节、人物、环境
5. **风格优先** - 当风格指南要求与原文结构冲突时，优先遵循风格指南（如短句优先于长句）

## 输出格式

必须在回复末尾使用以下格式：

<polish_result>
润色后：[润色后的完整文本]
</polish_result>

只输出润色后的文本和polish_result标签，不要解释修改了哪里。
