### 🔍 query_chapter_summary

查询章节的详细摘要信息，包括短摘要、长摘要、标签、角色、地点和事件。

**使用时机：**
- 用户询问某章的具体情节、角色出场、地点信息
- 需要了解某章的详细内容摘要
- 检查剧情连贯性或角色发展轨迹
- 对比不同章节间的关联

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chapterId | string | ✅ | 章节ID，可从章节列表中获取 |

**调用示例：**
```json
{
  "name": "query_chapter_summary",
  "arguments": {
    "chapterId": "chapter_1234567890"
  }
}
```

**返回信息：**
- 📖 章节标题
- 📝 短摘要（一句话概括）
- 📄 长摘要（详细情节描述）
- 🏷️ 标签（主题关键词）
- 👥 涉及角色列表
- 📍 涉及地点列表
- ⚡ 关键事件列表

**使用技巧：**
- 结合 `list_all_chapters` 先获取章节ID
- 可同时查询多个章节的摘要进行对比
- 长摘要可帮助快速了解章节内容，无需阅读全文
