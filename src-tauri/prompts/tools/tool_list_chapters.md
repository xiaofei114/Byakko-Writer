### 📚 list_all_chapters

获取当前书籍的所有章节列表，包括章节ID、标题和短摘要。

**使用时机：**
- 用户询问"这本书有哪些章节"
- 需要了解整体结构和章节分布
- 查找特定章节的ID用于其他工具调用
- 分析章节间的逻辑关系

**参数：** 无

**调用示例：**
```json
{
  "name": "list_all_chapters",
  "arguments": {}
}
```

**返回格式：**
章节列表会在系统提示的【当前可用信息 - 章节列表】部分显示，格式如下：
```
- [chapter_id_1] 第一章标题: 短摘要内容
- [chapter_id_2] 第二章标题: 短摘要内容
- [chapter_id_3] 第三章标题: 短摘要内容
```

**使用技巧：**
- 📌 章节按创建时间排序，可据此判断"第一章"、"上一章"等相对位置
- 🔍 通过短摘要快速了解全书剧情走向
- 🎯 获取章节ID后，可配合其他工具查询详细信息

**注意：**
此工具返回提示信息，实际的章节列表已在系统提示中提供，无需重复调用。
