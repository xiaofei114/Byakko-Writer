### 📖 query_chapter_content

获取章节的完整原文内容，用于深度分析、润色或引用。

**使用时机：**
- 用户要求查看某章的具体原文
- 需要对某段文字进行润色或修改建议
- 检查具体的对话、描写或细节
- 分析写作技巧或文风特点

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chapterId | string | ✅ | 章节ID |

**调用示例：**
```json
{
  "name": "query_chapter_content",
  "arguments": {
    "chapterId": "chapter_1234567890"
  }
}
```

**注意事项：**
- ⚠️ 返回内容可能较长，请根据需要处理
- 💡 如需了解大致内容，优先使用 `query_chapter_summary`
- 🎯 结合具体需求，针对性地分析内容

**典型应用场景：**
1. **润色建议**：获取原文后，针对具体段落给出修改建议
2. **剧情分析**：分析情节转折、伏笔设置等
3. **角色分析**：统计角色出场频率、分析角色言行一致性
4. **文风诊断**：检查是否存在重复用词、句式单一等问题
