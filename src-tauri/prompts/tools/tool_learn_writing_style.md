# 学习用户写作风格工具

## 工具名称
`learn_writing_style`

## 描述
分析用户的写作风格，生成风格指南供 AI 在后续创作中模仿用户的文风。

## 使用时机
- 用户明确要求"学习我的写作风格"、"分析我的文风"、"模仿我的风格"
- 用户说"按照我的风格写"但你还不了解其风格时
- 需要为用户提供风格一致的续写或润色时

## 参数

```json
{
  "bookId": "书籍ID（可选，不传则分析全局风格）",
  "force": false
}
```

## 使用示例

**场景1：用户要求学习风格**
```xml
<tool name="learn_writing_style">
{
  "bookId": "book_xxx"
}
</tool>
```

**场景2：用户要求强制重新分析**
```xml
<tool name="learn_writing_style">
{
  "bookId": "book_xxx",
  "force": true
}
</tool>
```

## 工作流程

1. **调用工具**学习用户写作风格
2. **等待结果**返回风格分析摘要
3. **告知用户**风格学习完成，后续创作将遵循该风格

## 注意事项

- 风格分析需要一定时间，请耐心等待
- 分析完成后，系统会自动保存风格配置
- 后续对话中 AI 会自动使用该风格进行创作
- 如果用户有多个书籍，可以分别学习每个书籍的风格
