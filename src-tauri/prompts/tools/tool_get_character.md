### 👤 get_character_card

获取单个角色卡的完整详细信息。

**使用时机：**
- 需要查看某个角色的完整设定
- **修改角色卡前，必须先调用此工具获取现有数据**
- 用户询问"某某角色的详细信息"
- 核对角色设定的一致性

**参数：**
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cardId | string | ✅ | 角色卡ID |

**调用示例：**
```json
{
  "name": "get_character_card",
  "arguments": {
    "cardId": "char_1234567890"
  }
}
```

**返回信息：**
- 🆔 角色ID
- 📛 姓名
- 🏷️ 别名列表
- ⚧ 性别
- 🎂 年龄
- 👤 外貌描述
- 🧠 性格特点
- 📖 背景故事
- 🎯 目标/动机
- 🔗 关系列表
- 🏷️ 标签
- 📝 备注
- 📅 创建/更新时间

**使用流程：**
```
用户要求修改角色
    ↓
调用 get_character_card 获取现有数据
    ↓
确认要修改的内容
    ↓
调用 update_character_card 更新
    ↓
展示更新后的角色卡
```

**提示：**
- 💡 获取角色卡后，可以向用户展示完整信息确认
- 🔄 修改前务必调用，避免覆盖已有数据
